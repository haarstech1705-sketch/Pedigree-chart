About / Privacy

👋 Introduction

PEDII is an educational pedigree drawing and learning tool for medical and biology students. It is designed for visualization, teaching, and practice, not for diagnosis or clinical decision-making.

💡 Project Background

This application was created to help users understand pedigree symbols, inheritance patterns, and chart structure in a simple, mobile-friendly interface.

📊 App Purpose

  • Learn pedigree chart concepts
  • Understand genetic inheritance patterns
  • Draw and visualize pedigree diagrams easily
  • Save or load diagrams as local files

⚙️ Development

This application has been built using modern web technologies with a focus on simplicity, clarity, and ease of use.

⚠️ Disclaimer

This app is strictly intended for educational purposes only. It does not diagnose, detect, prevent, or treat any disease or medical condition. Users should consult qualified healthcare professionals for any medical concerns.

🔒 Privacy & Data Use

  • No account is required.
  • No analytics or tracking are built in.
  • Save and load work through local files on the device.
  • Do not enter real patient-identifiable information.
  • For Play Store submission, publish a public privacy policy URL and link it in the store listing.

🎯 Objective

Through this project, the goal is to combine academic knowledge with technical skills to create a simple educational tool for pedigree learning.
